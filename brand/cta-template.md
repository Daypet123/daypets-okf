# 早安毛健康 · CTA 公版（2026-04-20 改版：無 H2 收尾段）

> 來源：6 份親改樣本歸納（第 12 階段）。
> 重大變更（2026-04-20）：CTA 從獨立 H2 改為**無 H2 的收尾段**。FAQ 之後直接以 1 段 `<p>` 結尾，不得插入任何 H2 標題。

## 公版模板（無 H2 收尾段）

```
以上是關於「{topic}」的全方位指南。歡迎持續追蹤《早安毛健康》，了解更多 {延伸主題1}、{延伸主題2}、{延伸主題3} 等寵物資訊，讓我們陪伴你與毛孩健康快樂地一起長大！
```

- `{topic}`：**直接使用當篇文章關鍵字原文**
- `{延伸主題1/2/3}`：從文章庫挑 3 篇延伸文章，須替換為實際文章名稱並加上超連結
- 除了 `{topic}` 與 `{延伸主題1/2/3}`，其餘文案固定，不得自行改寫

## 使用規則（硬規範）

- **不得在 CTA 段上方加任何 H2 標題**（例如 `<h2>CTA</h2>`、`<h2>推薦閱讀</h2>`、`<h2>結語</h2>` 一律禁止）
- FAQ 結束後，**直接**出現此收尾段 `<p>`
- 3 個延伸主題**必須加超連結**，連結到 `content-library.md` 中實際存在的 URL
- **不可編造不存在的文章連結**
- 不硬推商品（早安毛非電商）
- 品牌句「持續追蹤《早安毛健康》」**書名號必留**（final-html-validator 會偵測）
- `等寵物資訊` 為固定字樣，不再依題型切換

## 延伸主題挑選原則

1. **不得與正文內鏈重複**（硬規則）：CTA 的 3 個延伸主題不可選正文中已經出現過的內鏈文章。正文已連過的主題讀者已看到，CTA 應提供新的延伸方向，最大化內鏈覆蓋面
2. **優先鄰近物種**：寫布偶貓 → 優先選其他貓品種、幼貓照護，而非狗相關
3. **讀者旅程下一站**：寫品種介紹 → 可接其他相近品種、價格 / 花色 / 疾病
4. **3 個主題覆蓋不同面向**：避免 3 個都是同性質（例：不要同時連三篇貓品種百科）
5. **品質優先**：選文章庫中已發表、品質佳的文章，而非同主題的替代文
6. **不自我參照**：不連回本篇自己或同系列的重複文
7. **若使用者明示官方公版**：以使用者提供的 CTA 句型為最高準則

## 為何改為無 H2 收尾段（歷史紀錄）

- 2026-04-17 舊版：`<h2>CTA</h2>` 獨立成段
- 2026-04-20 Claire 裁決：6 份人工樣本一致把 CTA 寫成**無 H2 的收尾段**，歸屬 FAQ 之後直接接 `<p>`，不放標題
- 理由：CTA 加 H2 會讓它被誤判為「新章節」，打斷讀者流；無 H2 收尾段才是自然的結束語氣
