## 1. H3 命名風格（分三流派，看 H2 主題決定用哪個）

### 流派 A：（一）（二）（三）+ 問句／陳述句
- 適用：教學類、概念釐清類、比較類
- 範例：
  - 眼袋：`（一）眼袋是什麼？`、`（二）為什麼會有眼袋？5 大眼袋成因介紹`
  - 體脂：`（一）體脂肪是什麼？體脂肪功用為何？`、`（二）減肥不能只看體重？`

### 流派 B：主題符號 + 編號 + 名稱
- 適用：推薦類、清單類（景點、飯店、美食、產品）
- 範例：
  - 巴黎景點：`✨巴黎景點 1 凡爾賽宮 Palace of Versailles`、`✨巴黎景點 2 羅浮宮`
  - 巴黎飯店：`✦ 飯店 1 Paris Marriott Opera Ambassador Hotel`
  - 巴黎美食：`♥︎ 巴黎鐵塔餐廳 Madame Brasserie`
  - 巴黎必買：`★ 法國必買精品 Louis Vuitton 路易威登`
- 同一 H2 下，所有 H3 共用同一個符號
- 符號對照：
  - 景點／觀光 → ✨
  - 住宿 → ✦
  - 美食／餐廳 → ♥︎
  - 必買／推薦品 → ★
  - 步驟／要點 → 📍
  - 程序／SOP → 1️⃣ 2️⃣ 3️⃣ 或純數字
  - 一般子段 → 純 `（一）（二）`

### 流派 C：關鍵字 X 流汗衣服會臭的原因 1 / 2 / 3
- 適用：「同個名詞 + 編號」的橫向列表，如原因、步驟、特色
- 範例：
  - 流汗臭：`流汗衣服會臭的原因 1：衣物材質`、`原因 2：頻繁使用止汗劑`

## 2. 每個 H2／H3 內部一定要寫的東西

1. **撰寫方向**（淡藍字）
   - 不是完稿內容，是 brief
   - 句型：「從 XX 切入，帶到 YY，最後收到 ZZ」
   - 若是列點，直接寫列點（供撰文者展開）
2. **列點或表格**（展開用的骨幹資料）
   - 比較類、價格類、數值類 → 表格
   - 其餘 → bullet
3. **撰寫方式說明**（紫色字，可選）
   - 例：「以表格呈現」「以懶人包列點呈現」
4. **設計需求／配圖備註**（橘色字，可選）
   - 例：「配圖：三者加熱原理比較圖」「需客戶補：服務特色」
5. **帶入關鍵字**
   - 目標關鍵字（對應藍底）：該段至少 1 次
   - 長尾關鍵字（對應灰底）：該段至少 1 次
6. **段尾提醒**
   - 可插入「🚩以上僅供資訊確認，後續會經過改寫」

## 3. 早期 AI 常犯的架構錯誤（看完要避開）

| 錯誤 | 實際案例 | 正確作法 |
|------|---------|---------|
| 只有 H2、沒有 H3 | 「H2 #5 疾病解析（900 字）」 | 拆 H3：HCM、玻璃胃、PKD、多囊腎 |
| 每個 H2 只寫字數與「方向摘要」 | 「800 字，4 大花色 × 6 顏色圖鑑」 | 寫完整撰寫方向＋ H3 列點 |
| 把 intro 文字寫死在架構裡 | 02-structure.md 寫出完整 100 字「前言段」內容 | 架構只留規劃、intro 文字由 03-writer 生成（2026-04-20 翻轉） |
| 標題套話 | 「全解析／一次看／總整理／完整攻略」 | 換成「5 大照護重點解析／NO.1 推薦」 |
| H3 全用（一）（二） | 景點清單硬套（一）（二）（三） | 景點類用「✨主題 1」「✨主題 2」 |
| FAQ 全部用「Q1. Q2.」無內容 | 「Q3. 會掉毛嗎？」沒答 | 架構階段已先寫答的方向或列點 |
| 沒放收尾段 | 文末 FAQ 結束後直接斷掉 | 早安毛：FAQ 後接無 H2 收尾段（`<p>` 含品牌句 + 3 個延伸連結）；其他客戶：最後 H2 = CTA 段，先列產品特色再收 |

## 4. 實例對照：布偶貓（AI 版 → 人類版的差距）

| 項目 | AI 第一版 | 人類版（正確） |
|------|----------|---------------|
| H1 標題 | 布偶貓深度攻略｜花色、個性、HCM 心臟病、玻璃胃與台灣月花費全解析 | 布偶貓好養嗎？外觀、價格、平均壽命、照護要點全解析！ |
| 標題風格 | 功能條列、太學術 | 主問題 + 副標，用戶搜尋意圖語言 |
| 前言 | 有獨立前言段但和 Meta 重複堆疊 | H1 後不寫獨立 intro 段，直接接第一個 H2（2026-04-21 翻轉：編輯只需 H1、slug、meta） |
| H2 結構 | 11 個 H2 + 字數標註 | 7 個 H2，每個有 H3 展開 |
| H3 | 幾乎沒有 | 每個 H2 下 2–4 個 H3 |
| H2 #5 | HCM / 玻璃胃 / PKD 混寫一段 | 拆成「個性／缺點／遺傳疾病」三個 H3 |
| FAQ | 標題列 7 題，無內容 | 6 題都有 H3 題幹 |
| 結尾 | 「結語與延伸閱讀」 | 無單獨結語；FAQ 後即收 CTA |
