## 二、標題寫作模式（競品分析 + Welly 框架）

### 2.1 推薦類標題句型（structure-rules.md）

Welly 框架的推薦類句型：`XX 推薦！[年份]最新[N]款精選評比！`

### 2.2 競品高頻元素

| 元素 | 出現率 |
|------|--------|
| 含「推薦」 | 100% |
| 含年份 | 83% |
| 具體數字（N 款/N 大） | 78% |
| 評比/比較 | 44% |
| Dcard/PTT 佐證 | 22% |

### 2.3 Welly 框架禁止的標題套話

❌ 完整、一次看、總整理、一篇看懂、一次掌握

### 2.4 標題去 AI 感

- 善用情緒鉤子：口語反問、挑戰句、痛點切入
- 善用「｜」分隔主副標
- 不每篇套同一句型

## 三、文章架構分析

### 3.1 通用架構（競品 80%+ 共用）

```
1. 前言 → 2. 選購指南 → 3. 產品推薦 → 4. 比較表 → 5. FAQ → 6. 結語/CTA
```

### 3.2 早安毛專屬架構（structure-examples.md 第 6 節）

- H1 後直接接第一個 H2，不寫獨立 intro 段（2026-04-21 翻轉）
- 最後一個 H2 = FAQ 段
- FAQ 後接無 H2 收尾段 `<p>`（品牌句 + 3 個延伸連結）
- H3 符號預設 ✦（FAQ 用 Q：）
- 不列具體店家、不用價格表改行內文字

### 3.3 H2/H3 命名（structure-examples.md）

**H2**：
- 中文序號「一、二、三」
- 混合問句、驚嘆句、陳述句
- 可用「主題｜副標」或「問句？」格式

**H3 三種流派**：
- 流派 A（教學類）：（一）（二）（三）
- 流派 B（推薦清單類）：✦ 推薦 1 品牌名
- FAQ：Q：問題？

### 3.4 架構品質檢核表（15 條，structure-rules.md）

關鍵檢核項：
1. 標題 25～30 字
2. Meta Description 90～100 字
3. Slug 全小寫英文 dash
4. H2 ≥ 3 個
5. 非 FAQ 的 H2 每個至少 1 個 H3
6. FAQ ≥ 3 題
7. 不出現 H2 套話
8. H2 句型多樣性（不可全部問句）

## 四、寫作風格分析

### 4.1 語調定位（結合競品分析 + writing-rules.md）

```
學術嚴肅 ←────────────────→ 網路口語
       HOPET  臭味滾  毛孩時代  毛怪樂園  省錢小喵
                    ↑
              早安毛健康
              （部落格文體）
```

部落格文體判準（writing-rules.md）：
- 太口語 →「精神和胃口也會一起掉」→ 改「一起減弱」
- 太書面 →「存在顯著差異」→ 改「差很多」
- 座標：娛樂線、日常消費類的新聞稿

### 4.2 寫作六大技巧（quality-standards.md）

| 代號 | 技巧 | 推薦文適用場景 |
|:---:|------|--------------|
| T1 | 用已知比較未知 | 用飼主已知的乾糧經驗對比主食罐 |
| T2 | 用具體比喻抽象 | 「高含肉量罐頭就像鮮煮雞胸肉」 |
| T3 | 點出讀者相關性 | 附台灣的實際售價和購買通路 |
| T4 | 連結生活經驗 | 「你在寵物店看到的那面牆」 |
| T5 | 打臉讀者 | 「你以為無穀一定比較健康？」 |
| T6 | 破題解答困惑 | 「搞不懂主食罐和副食罐？」 |

### 4.3 段落寫作法（quality-standards.md + writing-rules.md）

**魚骨寫作法**：
- 魚頭（第一句）= 結論
- 魚骨+魚肉 = 數據、案例支撐
- 魚尾 = 重複強化

**金字塔法**：先講結論再補充

**段落硬規則**：
- ≤ 4 行（80～120 字）
- 每段 1 處粗體核心資訊
- 每段只做一件事：下判斷/補理由/給例子/提醒風險
- 每個非 FAQ 的 H2 至少 1 個 `<ul>/<ol>/<table>` 或 2 個 `<strong>`

### 4.4 句型多樣性（writing-rules.md）

- 混合短句（8～15 字）和長句（20～35 字）
- 不連續 3 句同長度
- 不連續 2 段相同開頭
- 反問句每 300～500 字 1 次
- 「不是…而是…」≤ 2 處/篇
- 口語助詞（唷/呢/了/喔）每 400～600 字 1 次
- 轉折詞池輪替（不過、然而、只不過、此外、同時……）同一 H2 內同一轉折詞不得連用 2 次

### 4.5 去 AI 感清單（writing-rules.md）

**必刪句型**：
1. 預告型：「接下來要介紹…」「先搞清楚…免得…」
2. 重述型：前面已傳達過同樣資訊
3. AI 貼心提醒：「這樣你就能知道…」「記住這幾點就夠了」

**判準**：把該句遮起來，上下文仍通順、資訊沒缺 → 刪掉

**避免 AI 排比**：三件事並列，前後兩件完整句、中間改短片語

## 五、標點符號完整規則（punctuation-rules.md）

核心要點：
- 逗號「，」隔分句；頓號「、」隔並列詞
- 三項以上並列最後一項前用「和/及/或」，不加頓號
- 省略號 = 6 點「……」（❌ `...` `。。。` `⋯`）
- **禁止使用破折號「——」**，改用逗號、冒號或句號重組
- 嘆號全篇 ≤ 5 個
- ❌ 禁止「？！」「！？」
- 數字區間用「～」（18～20 小時）
- 中文與英文/數字之間加半形空格

## 六、豐富性原則（richness-principles.md）

### 6.1 Parity + Gap-Fill

- **Parity**：SERP 前 3 名都寫的 H2 = 我方必須涵蓋（換角度、換句序、換實例）
- **Gap-Fill**：競品沒寫但重要的主題（來自 PAA、Related Searches）
- Gap-Fill 上限：不超過總 H2 的 40%

### 6.2 量化數字下限

- 「該多久/多少量/多少錢」→ 必須給具體數字或區間
- 價格段落至少 3 個分級

### 6.3 具名資源

- 外部資源必須用真實名稱 + 超連結
- ❌「某平台」「一般來說」「通常」→ 改寫為具體

## 七、各產品類別專業知識

### 7.1 貓飼料
- 核心指標：蛋白質 30～40%+、動物性蛋白優先、碳水 < 25%
- 營養素：牛磺酸、脂肪、維生素 A/D
- 避雷：肉類副產品、4D 動物成分
- 認證：AAFCO、NRC、WDJ
- 分類維度：穀類（無穀/低穀/含穀）、年齡（幼貓/成貓/熟齡貓）、機能（室內/腎臟/腸胃）

### 7.2 貓罐頭
- 主食罐 vs 副食罐區別（AAFCO + NRC 雙重認證）
- 質地：肉絲型、泥狀/慕斯型、湯罐型、肉塊型
- 膠類爭議：卡拉膠、瓜爾膠、三仙膠
- 關鍵成分：牛磺酸、鈣磷比、必需脂肪酸

### 7.3 貓砂
- 種類：礦砂、豆腐砂、木薯砂、花生砂、木屑砂、水晶砂、紙砂
- 評比維度：凝結速度、除臭效果、粉塵、可否沖馬桶、重量
- 砂盆數量公式：貓咪數量 + 1
- 換砂方法：漸進式（新舊砂混合比例逐步調整）

### 7.4 貓肉泥
- 定位：零食非主食（除非標示主食級）
- 餵食：每週 2～3 次，每日不超過 2 條
- 選購：新鮮肉類為主、無人工添加、低磷低鈉、碳水 < 5%
- 用途：補水、誘食、訓練獎勵

## 八、早安毛 vs 競品的差異化機會

### 8.1 競品常見問題（早安毛應避免）

1. 自家產品永遠排第一、評分最高
2. 缺點避重就輕（自家只寫優點）
3. 只寫「蛋白質高」但不附數字
4. 同品牌多篇文章段落幾乎一模一樣
5. 過度堆砌關鍵字

### 8.2 早安毛的差異化武器

1. **中立評測**：優缺點並陳（競品做不到）
2. **數據導向**：附具體營養成分數字
3. **情境推薦**：依需求分流（腎貓/多貓/預算）
4. **社群佐證**：整合 Dcard/PTT 真實觀點
5. **持續更新**：標示更新日期

## 九、Google 官方優質評論指南摘要

來源：[Google Search Central — 撰寫優質評論](https://developers.google.com/search/docs/specialty/ecommerce/write-high-quality-reviews?hl=zh-tw)（2025-12-18 更新）

### 9.1 Google 要求的評論品質要素

| 要素 | 說明 | 早安毛對應做法 |
|------|------|--------------|
| 使用者視角 | 從消費者角度評估 | 以飼主需求為出發點，非品牌方 |
| 展現專業 | 證明對評論內容瞭若指掌 | 附具體成分數字、AAFCO/NRC 認證說明 |
| 第一手證據 | 影像、音訊、連結佐證 | 引用社群真實討論、開箱影片連結 |
| 量化評估 | 各方面成效的量化資料 | 蛋白質 %、碳水 %、價格區間 |
| 競品差異 | 說明類似產品的差異 | 比較表 + 情境推薦分流 |
| 優缺點並陳 | 基於原創研究 | 每個推薦產品都列注意事項 |
| 決策因素 | 點出最重要的選購考量 | 選購指南段落（挑選重點） |
| 多賣家連結 | 提供多個購買管道 | 早安毛不列具體店家，但提及「各大電商平台」 |
| 排名理由 | 最佳推薦需說明原因 | 每個推薦都有「適合族群」和定位描述 |
| 獨立說明 | 每個產品都有足夠說明 | 每個 H3 推薦區塊完整獨立 |

### 9.2 與競品分析交叉對照

競品常違反的 Google 指南：自家產品排第一但無客觀理由（違反「排名理由」）、缺點避重就輕（違反「優缺點並陳」）、缺乏量化數據（違反「量化評估」）。早安毛的中立定位天然符合 Google 的評論品質要求。
