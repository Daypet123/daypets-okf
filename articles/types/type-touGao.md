# 投稿小文（touGao）— 類型規範

## 定位

改寫原文為新聞稿形式，投稿至寵物媒體或生活類媒體。目的是以第三方媒體的角色傳遞品牌資訊，提升公信力。

## 基本參數

| 項目 | 規範 |
|------|------|
| 字數 | 500～850 字 |
| 人稱 | 無第一人稱，中性第三人稱 |
| 口吻 | 類新聞報導、生活化但保持客觀、可引用專家觀點 |
| 段落長度 | 每 3～4 行分段，單段不超過 6 行 |
| 結構 | 前言（50～100 字）→ H2 段落 × 3 → 結語 → 推薦 |
| 連結 | 文中自然置入錨文字連結 |

## 口吻指引

### 要做到的
- 中性客觀的報導語氣
- 可使用「飼主」等第三方泛稱
- 「獸醫師指出」「根據調查」等引述型表述，僅限 research brief 中有對應來源時使用
- 數據引用僅限 research brief 中有明確出處的資料；無來源時改用描述性語言（「不少飼主」「常見的情況是」）
- 段落之間邏輯清晰、過渡自然

### 要避免的
- 絕對不可出現「我」「你」「我們」等直接稱呼
- 不可有促銷語氣（「快來購買」「限時優惠」）
- 不可有過度行銷感的結語
- 偶爾用「飼主」替代「你」是可以的

## 結構模板

```
# [新聞稿標題，有資訊價值感]
slug: [english-slug]

[前言：50～100 字，交代背景或切入點，建立閱讀動機]

## [H2-1：資訊型段落標題]
[段落內容：核心資訊、數據、專家觀點]

## [H2-2：資訊型段落標題]
[段落內容：延伸資訊、實用建議]

## [H2-3：資訊型段落標題]
[段落內容：補充面向或行動建議]

[結語：溫和收束，不促銷]
更多寵物健康資訊，可參考 早安毛健康 相關文章。
```

## 標題風格範例

- ✅ 換季是狗狗的照護考驗！出現五種狀況請小心
- ✅ 獸醫師提醒：貓咪初老五大徵兆，飼主別忽略
- ✅ 台灣飼主最常忽略的寵物換季照護三件事
- ❌ 我家狗狗換季好辛苦（有第一人稱）
- ❌ 超重要！五個你一定要知道的狗狗照護秘密（標題黨）

## 投稿媒體風格參考

### 妞新聞
- 稱呼讀者為「妞妞們」
- 語氣活潑、帶少量驚嘆號和「～」
- 常用「快一起看看！」「是不是也覺得……」
- 字數：約 800～1400 字

### 寵物部落
- 語氣偏專業但親切
- 常引用獸醫師、學者、研究
- FAQ 格式出現在文末
- 有延伸閱讀連結
- 字數：約 1000～2200 字
