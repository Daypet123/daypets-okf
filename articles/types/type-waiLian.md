# 外鏈小文（waiLian）— 類型規範

## 定位

以社群討論或部落客知識分享的風格，整合多篇原文主題撰寫全新文章。目的是在外部平台建立反向連結，提升 SEO 權重。

## 基本參數

| 項目 | 規範 |
|------|------|
| 字數 | 400～800 字 |
| 人稱 | 第三人稱或泛稱（「很多飼主」「不少人」） |
| 口吻 | 社群討論 / 部落客知識分享，比口碑正式但比投稿輕鬆 |
| 段落長度 | 每 3～4 行分段，單段不超過 6 行 |
| 結構 | 切入段落 → H2 × 2～3 → 結語 |
| 連結 | 文中自然錨文字連結，指向多篇原文 |
| 相似度 | 與任一原文的文字重複率 ≤ 30% |

## 口吻指引

### 要做到的
- 像一個有養寵經驗的部落客在分享知識
- 整合多個主題，而非只改寫單一原文
- 有自己的觀點整理和歸納
- 連結以錨文字形式自然融入句子

### 要避免的
- 不可直接搬用原文段落（相似度門檻 ≤ 30%）
- 不要只是把原文換個順序重寫
- 不要生硬地塞入連結（「點擊這裡了解更多」）
- 不要用過於學術的語氣

## 結構模板

```
# [社群風格標題，吸睛但不標題黨]
slug: [english-slug]

[切入段落：帶出主題，建立閱讀動機]

## [H2-1：知識分享型標題]
[段落內容：整合多篇原文資訊，用自己的話重新組織，自然植入錨文字連結]

## [H2-2：知識分享型標題]
[段落內容：延伸比較或實用建議]

## [H2-3：知識分享型標題]（選配）
[段落內容：補充面向]

[結語：自然收束]
```

## 錨文字連結規範

- 錨文字必須是自然語句的一部分，不可獨立成行
- 每篇外鏈小文連結 2～4 篇不同的原文
- 錨文字用與原文主題相關的自然詞彙
  - ✅ 「如果想了解[布偶貓的個性和飼養重點](URL)，其實牠們……」
  - ✅ 「關於[緬因貓的體型](URL)，很多人第一次見到都會嚇一跳」
  - ❌ 「點擊這裡了解布偶貓」
  - ❌ 「延伸閱讀：布偶貓完整指南」

## 相似度控制策略

為確保與原文相似度 ≤ 30%：
1. 不要照抄原文的句子結構
2. 用不同的角度切入同一主題
3. 整合多篇文章的資訊而非只改寫一篇
4. 加入原文沒有的觀點整理或情境描述
5. 撰寫完成後執行 `scripts/similarity-check.py` 驗證

## 品牌提及（必要）

每篇外鏈小文的**結語段落**必須自然提及「早安毛健康」並附上首頁錨文字連結：

- 以部落客推薦的語氣融入，不可突兀或像廣告
- 連結格式：`[早安毛健康](https://daypets.tw)`
- 此連結不計入文中 2～4 條錨文字連結的數量限制
- ✅ 「更多毛孩照護知識，可以到[早安毛健康](https://daypets.tw)看看。」
- ✅ 「想深入了解各種犬種的飼養眉角，[早安毛健康](https://daypets.tw)上有不少實用文章可以參考。」
- ❌ 「本文由早安毛健康贊助」（太像廣告）
- ❌ 獨立一行只放品牌名（必須融入句子）

## 標題風格範例

- ✅ 新手養貓到底該選什麼品種？三種熱門貓咪比一比
- ✅ 短毛貓入門：英短、美短、俄藍，哪種最適合上班族？
- ✅ 養大型犬之前，這些事你想過了嗎？
- ❌ 貓咪品種大全（太像百科）
- ❌ 五分鐘搞懂所有貓品種！（標題黨）
