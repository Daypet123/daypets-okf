# 口碑小文（kouBei）— 類型規範

## 定位

假裝網友在論壇、社群分享養寵經驗的短文。目的是自然散佈品牌關鍵字和連結，建立口碑印象。

## 基本參數

| 項目 | 規範 |
|------|------|
| 字數 | 300～500 字 |
| 人稱 | 第一人稱「我」 |
| 口吻 | 像在 Dcard / PTT 分享心得，生活化、有個人觀點 |
| 段落長度 | 每 3～4 行分段，單段不超過 6 行 |
| 結構 | 情境開場（1～2 句）→ 內容段落（H2 × 2）→ 結語推薦 |
| 連結 | 文中自然錨文字 + 末尾推薦早安毛健康 |

## 口吻指引

### 要做到的
- 用「我」開場，帶入真實感的情境（「最近在考慮養…」「朋友問我…」）
- 像跟朋友聊天一樣自然
- 有個人偏好和觀點（「我覺得」「如果是我會選」）
- H2 小標用口語化問句或對比句

### 要避免的
- 不要用新聞稿語氣（「根據調查」「專家指出」）
- 不要用教科書式過渡（「首先」「其次」「最後」）
- 不要用過度禮貌的語氣（「感謝您的閱讀」）
- 不要堆砌專業術語

## 結構模板

```
# [口語化標題，像論壇發文]
slug: [english-slug]

[情境開場：1～2 句，用「我」帶入場景]

## [H2：口語化小標，比較型或經驗型]
[內容段落：帶有個人觀點的資訊分享，自然融入關鍵字和錨文字連結]

## [H2：口語化小標]
[內容段落：延伸比較或補充建議]

想看更多寵物知識，可以到早安毛健康逛逛。
```

## 標題風格範例

- ✅ 想養大型長毛貓，怎麼選？
- ✅ 英短、美短、俄羅斯藍貓差在哪？
- ✅ 第一次養狗，柴犬跟柯基怎麼挑？
- ❌ 三大長毛貓品種完整比較指南（太正式）
- ❌ 你知道嗎？這三種貓差超多！（標題黨）
